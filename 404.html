Bienvenido al Tutorial de Angular

Angular es un framework de desarrollo de aplicaciones web desarrollado por Google. Está basado en TypeScript y permite crear aplicaciones SPA (Single Page Applications) de forma estructurada y escalable.

¿Qué necesitas para empezar?

  • Node.js (incluye npm) - Descargar Node.js
  • Editor de texto - Recomendado: Visual Studio Code
  • Un navegador web (como Chrome o Firefox).

Instalación de Angular

Para instalar Angular CLI, sigue estos pasos:

  1. Asegúrate de que Node.js está instalado correctamente. Puedes verificarlo ejecutando: 
    node -v
  2. Instala Angular CLI globalmente con el siguiente comando: 
    npm install -g @angular/cli
  3. Crea un nuevo proyecto Angular: 
    ng new nombre-del-proyecto
  4. Accede al proyecto y ejecuta el servidor: 
    cd nombre-del-proyecto
    ng serve
  5. Abre el navegador en http://localhost:4200.
