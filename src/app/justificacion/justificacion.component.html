Justificación de elección

Angular es un framework de desarrollo de aplicaciones web desarrollado por Google. Está basado en TypeScript y permite crear aplicaciones SPA (Single Page Applications) de forma estructurada y escalable.

→ ¿Por qué elegir este framework?

Permite la creación de interfaces de usuario complejas y de alto rendimiento, facilitando la gestión de datos y la interacción con el servidor.

→ ¿Qué aplicaciones tiene?

Angular es ideal para crear aplicaciones web interactivas, como:

  • Sistemas de gestión de contenido (CMS).
  • Aplicaciones de comercio electrónico.
  • Aplicaciones de red social.
  • Aplicaciones móviles híbridas.
  • Paneles de administración y dashboards.

→ ¿Sobre qué lenguajes se apoya Angular?

Angular se apoya principalmente en:

  • TypeScript: Un superset de JavaScript que agrega tipado estático, lo que permite un código más seguro y fácil de mantener.
  • HTML: Para estructurar el contenido y las vistas de la aplicación.
  • CSS: Para aplicar estilos y mejorar la presentación visual de la aplicación.
