Primeros Pasos

→ Cómo realizar una primera aproximación en Angular ("Hola Mundo")

¿Qué necesitas para empezar?

Para realizar una primera aproximación con Angular, necesitarás instalar las herramientas básicas:

  • Node.js: Descárgalo desde nodejs.org.
  • Angular CLI: Una herramienta que facilita la creación y gestión de proyectos Angular.

→ Pasos para crear un proyecto "Hola Mundo"

  1. Instalar Angular CLI:

    Abre la terminal y ejecuta el comando:

    npm install -g @angular/cli
  2. Crear un nuevo proyecto:

    Ejecuta el siguiente comando en la terminal:

    ng new hola-mundo

    Sigue las instrucciones en pantalla, seleccionando la configuración predeterminada.

  3. Navegar al directorio del proyecto:

    Accede a la carpeta recién creada con:

    cd hola-mundo
  4. Iniciar el servidor de desarrollo:

    Ejecuta el siguiente comando para arrancar el servidor:

    ng serve

    Esto iniciará un servidor de desarrollo y abrirá la aplicación en tu navegador en http://localhost:4200.

  5. Modificar el componente inicial:

    Abre el archivo src/app/app.component.html en tu editor de código y reemplaza su contenido con:

    <h1>Hola Mundo desde Angular</h1>

    Guarda los cambios y actualiza el navegador para ver el resultado.

¿Qué obtendrás?

Al completar estos pasos, habrás creado tu primera aplicación Angular que muestra el mensaje "Hola Mundo" en el navegador. Esto sirve como base para desarrollar aplicaciones más complejas.
