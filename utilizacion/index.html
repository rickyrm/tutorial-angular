Utilización

Utilización

→ ¿Cómo empezar a crear una aplicación base en Angular?

1. Crear una aplicación base

  1. Generar un nuevo proyecto:

    En la terminal, ejecuta:

    ng new mi-aplicacion

    Sigue las instrucciones para configurar el proyecto (puedes aceptar las opciones predeterminadas).

  2. Acceder al directorio del proyecto:

    Una vez creado el proyecto, navega al directorio con:

    cd mi-aplicacion
  3. Iniciar el servidor de desarrollo:

    Ejecuta el comando:

    ng serve

    Esto lanzará un servidor de desarrollo y abrirá la aplicación en tu navegador en http://localhost:4200.

3. Modificar el componente inicial

Abre el archivo src/app/app.component.html en tu editor de código y reemplaza su contenido con el siguiente código:

<h1>¡Bienvenido a mi primera aplicación Angular!</h1>

Guarda los cambios y observa la actualización automática en tu navegador.

→ ¿Cómo visualizar los resultados de la ejecución?

Visualización de la aplicación

  1. Abre tu navegador web.
  2. Dirígete a la URL proporcionada por el servidor de desarrollo, generalmente http://localhost:4200.
  3. Verás la aplicación base ejecutándose con el mensaje personalizado que configuraste.

Actualización en tiempo real

Angular CLI incluye una funcionalidad llamada Hot Module Replacement (HMR), que actualiza automáticamente la página en el navegador cada vez que guardas un cambio en los archivos del proyecto.

→ ¿Qué utilidades tiene Angular?

Principales utilidades del framework Angular

  • Desarrollo de aplicaciones web dinámicas: Angular permite crear aplicaciones con funcionalidades avanzadas, interactividad y estructuras de datos dinámicas.
  • Estructura modular: Facilita la organización del código en módulos, componentes y servicios, lo que mejora la escalabilidad y mantenibilidad del proyecto.
  • Data binding bidireccional: Sincroniza automáticamente el modelo de datos con la vista, reduciendo la cantidad de código necesario para manejar actualizaciones de la interfaz.
  • Inyección de dependencias: Permite gestionar eficientemente servicios y recursos compartidos en la aplicación.
  • Herramientas avanzadas: Incluye un sistema de enrutamiento, validación de formularios, animaciones y soporte para pruebas unitarias y de integración.
  • Soporte para Progressive Web Apps (PWA): Angular facilita la creación de aplicaciones web que se comportan como aplicaciones nativas.
