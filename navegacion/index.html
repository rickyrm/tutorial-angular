Navegacion

Enrutamiento y Navegación en Angular

Angular ofrece un sistema robusto para gestionar la navegación entre vistas en aplicaciones de una sola página (SPA). Este sistema se basa en el uso del **Router** y las **Rutas** para determinar qué componente se debe mostrar según la URL actual.

¿Qué es el Enrutamiento?

El enrutamiento en Angular es un mecanismo que permite a los usuarios navegar por diferentes vistas de la aplicación. Se logra configurando rutas en el archivo app-routing.module.ts y usando el componente RouterOutlet.

Configurando el Enrutamiento

Los pasos básicos para configurar el enrutamiento en una aplicación Angular son los siguientes:

  1. **Importar el Módulo de Enrutamiento**: Angular incluye un módulo llamado RouterModule que se utiliza para configurar las rutas.
  2. **Definir las Rutas**: En el archivo app-routing.module.ts, defines las rutas de tu aplicación.
  3. **Agregar el RouterOutlet**: Este componente se utiliza en el archivo HTML para indicar dónde se cargarán las vistas.

Ejemplo de Configuración

Un ejemplo básico de configuración de rutas podría ser:

          
      import { NgModule } from '@angular/core';
      import { RouterModule, Routes } from '@angular/router';
      import { HomeComponent } from './home/home.component';
      import { AboutComponent } from './about/about.component';
      
      const routes: Routes = [
        { path: '', component: HomeComponent }, // Ruta por defecto
        { path: 'about', component: AboutComponent } // Ruta a "About"
      ];
      
      @NgModule({
        imports: [RouterModule.forRoot(routes)],
        exports: [RouterModule]
      })
      export class AppRoutingModule { }

Componentes Clave para el Enrutamiento

Angular proporciona varios componentes y directivas útiles para el enrutamiento:

  • RouterOutlet: Marca el lugar donde se renderizará el componente asociado a una ruta.
  • routerLink: Se utiliza en las vistas HTML para crear enlaces de navegación.
  • routerLinkActive: Permite aplicar estilos dinámicos a enlaces que estén activos.

Navegación entre Vistas

La navegación entre vistas se hace mediante enlaces o programáticamente desde el código TypeScript. Ejemplo de un enlace HTML: 

          
      <a routerLink="/about" routerLinkActive="active">Ir a About</a>

Navegación Programática

También puedes navegar entre rutas usando el servicio Router en tu código TypeScript:

          
      import { Router } from '@angular/router';
      
      constructor(private router: Router) { }
      
      navegar() {
        this.router.navigate(['/about']);
      }

Protegiendo Rutas con Guards

Angular incluye una funcionalidad llamada **Guards** para proteger rutas. Un ejemplo común es evitar que usuarios no autenticados accedan a ciertas páginas: 

          
      import { Injectable } from '@angular/core';
      import { CanActivate } from '@angular/router';
      
      @Injectable({
        providedIn: 'root'
      })
      export class AuthGuard implements CanActivate {
        canActivate(): boolean {
          return isLoggedIn(); // Lógica de autenticación
        }
      }
